# Теги в HTML
 
 структура парного тега:
 **<тег>тело<тег>**

 Пример:
 ```html
 <h1>Это заголовок h1</h1>
 ```

Так же были изучены:

* `<section>`
* `<form>`
* `<article>`
* `<main>`
* `<header>`

 ***

# Базовые HTML-теги

Примеры:
```html
<h2>Заголовок</h2>
<p>Абзац текста</p>
<strong>Жирный</strong>
<em>Курсив</em>
<hr>
<a href="https://example.com"></a>
<img src="example.jpg" alt="Описание">
```
